#vd

### First Setup
##### mac
alias pyt3=/Library/Frameworks/Python.framework/Versions/3.9/bin/python3
alias pip3=/Library/Frameworks/Python.framework/Versions/3.9/bin/pip3
```sh
pyt3 --version
pip3 --version
pyt3 -v
exit()
```
sudo pip3 install -r requirements.txt
##### unix
```sh
python3 --version
pip3 --version
```
sudo pip3 install -r requirements.txt


### Run
##### mac
pyt3 main.py
##### unix
python3 main.py
